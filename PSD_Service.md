1. Общее описаниеAntigravity PSD Service — это внутренний модуль платформы «Рутина Хаб», предназначенный для автоматической модификации текстовых слоев в заранее подготовленных шаблонах .psd.Модуль работает локально на сервере, читает исходный файл, рекурсивно находит целевой текстовый слой по его имени, подменяет строковые данные и генерирует новый .psd файл для отдачи пользователю.2. Стек и зависимостиСреда выполнения: Node.js (рекомендуется v16+)Основная библиотека: ag-psd (чтение и запись структуры PSD)Встроенные модули: fs (работа с файловой системой)Команда для установки зависимостей:Bashnpm install ag-psd
3. Требования к PSD-шаблонамДля корректной работы модуля дизайнеры должны соблюдать следующие правила при создании шаблонов:Именование: Целевой текстовый слой должен иметь уникальное и предсказуемое имя на латинице (например, TaskTitle, UserName).Тип текста: Использовать Paragraph Text (текст в рамке), а не Point Text. Это предотвратит выход длинного текста за границы композиции — текст будет переноситься на новые строки в пределах заданного контейнера.Шрифты: Шрифты, используемые в шаблоне, должны быть стандартными или прилагаться к проекту, чтобы у конечного пользователя не возникло проблем при открытии файла.4. API функции updateTextLayerОсновная функция модуля отвечает за поиск и замену текста.Входные параметры:ПараметрТип данныхОписаниеlayersArrayМассив слоев (обычно psd.children), в котором производится поиск.targetLayerNameStringТочное имя слоя в Photoshop, который нужно изменить.newTextStringНовая текстовая строка для вставки.Возвращаемое значение:Boolean — true, если слой был успешно найден и изменен; false, если слой с таким именем не найден.5. Пример использования (Интеграция)JavaScriptconst fs = require('fs');
const { readPsd, writePsd } = require('ag-psd');

// Инициализация
const templateBuffer = fs.readFileSync('path/to/template.psd');
const psdData = readPsd(templateBuffer);

// Параметры задачи из Рутина Хаб
const layerName = 'TaskTitle';
const taskText = 'Завершить документацию Antigravity';

// Вызов функции (предполагается, что функция updateTextLayer определена)
const isSuccess = updateTextLayer(psdData.children, layerName, taskText);

if (isSuccess) {
    const outputBuffer = writePsd(psdData);
    fs.writeFileSync('path/to/output.psd', outputBuffer);
    console.log('Antigravity: Файл успешно сгенерирован.');
} else {
    console.warn(`Antigravity Ошибка: Слой ${layerName} не найден в шаблоне.`);
}
6. Известные ограничения (Технический долг)⚠️ Внимание: Разработчикам и менеджерам продукта необходимо учитывать следующие ограничения библиотеки ag-psd:Отсутствие рендеринга шрифтов: Модуль изменяет только метаданные текста, но не перерисовывает его растровое превью (пиксельный кэш), так как не имеет встроенного движка Adobe.Поведение у конечного пользователя: При открытии сгенерированного файла в Photoshop пользователь может увидеть старый текст. На слое появится предупреждающий значок (желтый треугольник). Для применения изменений пользователю потребуется дважды кликнуть по миниатюре текстового слоя.Невозможность прямой конвертации: Сгенерированный .psd файл нельзя напрямую конвертировать в .jpg или .png на стороне сервера (например, через ImageMagick) — на экспортированной картинке отобразится старый текст.
